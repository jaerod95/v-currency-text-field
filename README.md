# v-currency-text-field Currency text field for Vue.js

## Features

- Lightweight
- Mobile support
- Component or Directive flavors
- Accept copy/paste
- Editable

## Usage

### Globally

```js
import Vue from 'vue'
import currency from 'v-currency-text-field'

// register component <v-currency-text-field>
Vue.use(currency)
```
